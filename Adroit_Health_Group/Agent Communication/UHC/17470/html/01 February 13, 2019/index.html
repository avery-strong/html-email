UHC Checkout Validation Enhancements

Attention Agents,

In order to submit UHC applications online more efficiently, several system enhancements have been made. Please review the updates outlined below prior to your next submission.

1.

 

Validate first and last name.

These fields cannot contain a number.

2.

 

Validate the residential address.

This is required.

3.

 

A clickable link has been added that will copy the mailing address to the residential address.

4.

 

Validate that the residential address starts with a number

Example: 123 Elm is valid, Elm 123 is not

5.

 

Validate that the residential address does not include PO Box.

This is not acceptable for the address field.

6.

 

The ”Return To Cart” button has been removed from the application page (listed below the ‘send e-signature’ section. PLEASE NOTE: be sure that your quote is accurate before moving forward to the application page!)

If you have any questions regarding this update, please contact agentservices@adroithealthgroup.com.
